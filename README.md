# 地球スピナー / Earth Spinner

ブラウザで動く球体型ハンドスピナー。ドラッグで自由軸回転し、ベアリング風の慣性でひたすら回ります。外部ライブラリなしの単体 HTML（Canvas 2D のみ）。

**公開URL**: https://3d-spinner.ryo-email.workers.dev

## 特徴

- **疑似3D球体**：Canvas 2D でフロント半球をピクセル単位にテクスチャマッピング
- **地球テクスチャ**：地球写真（正距円筒）をデフォルト表示。外部URL読込／ローカル画像アップロードにも対応
- **自由軸回転**：クォータニオンで姿勢を保持（ジンバルロック回避）、ドラッグ1:1追従
- **慣性・加速**：離した勢いで回転継続、同方向にドラッグを繰り返すと加速（摩擦減衰つき）
- **発光エフェクト**：速度が上がるほど大気ハロー・リム・HUD が輝く
- **入力**：Pointer Events でマウス／タッチ統一、レスポンシブ対応
- **速度表示**：RPM 換算の数値＋バー

## 使い方

起動時にデフォルトの地球写真（`earth.jpg`）を表示します。左上パネルから別画像のURL読込・ローカル画像アップロード・「地球に戻す」（デフォルトへ復帰）ができます。

> 画像URL読込は CORS 対応画像のみ（`getImageData` で参照するため）。読めない場合はローカルアップロードを使ってください。
> デフォルトの地球写真は Wikimedia Commons の正距円筒画像（NASA 由来 / パブリックドメイン）を使用。
